La struttura dei contenuti

CARTELLA SRC

Nella cartella src si trovano i contenuti da elaborare per il riutilizzo del pacchetto Unibo Application Theme.

Le cartelle sono divise in:

  • assets: che a sua volta contiene:
    • img: cartella dove posizionare tutte le immagini statiche
    • js: cartella dei javascript che verranno poi utilizzati dal template ed il file che li incorpora tutti, app.js
    • scss: cartella con tutti i file scss. In essa i file sono stati suddivisi per macro-funzionalit√†. In seguito vediamo il suo funzionamento nel dettaglio
    • webfonts: troviamo tutti i file che riguardano i font, suddivisi nelle rispettive cartelle
  • layouts: in questa cartella troviamo tutti i template di base che vengono richiamati dalle pagine. Nell'attuale pacchetto troviamo il file default, che viene utilizzato come base per tutte le pagine. Il file styleguide, invece, viene utilizzato per la creazione delle pagine che si trovano all'interno della cartella styleguide e che quindi servono alla creazione delle linee guida per lo stile dei singoli elementi del pacchetto.
  • pages: all'interno di questa cartella ci sono tutte le pagine create per questo tema e una cartella styleguide contenente le pagine che invece sono da guida per la stilizzazione dei singoli elementi.
  • partials: la cartella contiene tutti gli snippet di codice da richiamare nelle pages. All'interno della cartella troviamo ulteriori cartelle, come ad esempio foundation, che racchiude tutti gli snippet di foundation, form, che contiene tutti gli snippet relativi alle form, responsive table, che contiene tutti gli snippet relativi alle tabelle responsive.
  • template-list: in questa cartella troviamo il file template.html che serve a scrivere la struttura della pagina iniziale, dove visualizzare tutti i template disponibili e il file md che viene inglobato e letto da template.html.
